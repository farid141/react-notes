# UseRef

Digunakan untuk menyimpan nilai yang tidak memicu re-render ketika berubah, dan juga untuk mengakses elemen DOM secara langsung. Menghasilkan reference object dapat berupa:

- Element DOM
- Obj tidak hilang ketike re-render (interval, timeout) seperti kita memberi perintah ke browser dan akan dilakukan terus menerus setelah perintah diberikan.

```jsx
edf Player(){
 const playerName = useRef()
 return (
  <input ref={playerName.current}> // akses ref
  <button onClick={()=>playerName.current.value='test'}>set name</button>
 )
}
```

## Attribut Element

Atribut untuk mengakses element.

## React 19 Kebawah

untuk react 19.0 kebawah, kita tidak bisa mempassing langsung ref dengan props, kita harus membungkusnya dengan forwardRef.

```jsx
const Input = forwardRef((props, ref) => {})
```
