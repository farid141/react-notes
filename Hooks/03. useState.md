# useState

Digunakan untuk membuat state yang dapat mentrigger re-render komponen tempat didefinisikan dan juga childnya ketika nilainya terjadi. Untuk mengubah state dipanggil fungsi setter.

Contoh callback function untuk `onChange` berisi perintah untuk mengubah state, maka re-render ditandai dengan element yang menggunakan state tersebut berubah nilai.

Dalam satu callback, semua perintah ubah state dikumpulkan dan dieksekusi bersamaan ketika re-render.

## Setter dengan Callback

Sebelumnya dijelaskan bahwa semua perintah ubah state akan ditampung. Selain itu, nilai yang dievaluasi juga belum berubah. Akibatnya apabila kita buat dua increment state, perubahan pertama akan diabaikan.

Agar perubahan tidak diabaikan, berikan argumen setter dengan callback yang memiliki param `prev_state`.
`setActive(current => !current)`
