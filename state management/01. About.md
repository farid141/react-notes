# About

State dibagi menjadi tiga berdasarkan scopenya:

1. local state: state dibuat dalam komponen langsung
2. component state: useContext() mencegah prop-drilling
3. app-wide state: tambahan library (redux dan redux-toolkit)

## Kapan Menggunakan State Global?

- Autentikasi pengguna: informasi login, token, roles, dsb. digunakan di banyak tempat (header, sidebar, dashboard, dsb.).
- Tema dan preferensi pengguna: dark/light mode, bahasa, layout, dsb.
- Keranjang belanja (cart): data produk yang tersebar di berbagai komponen halaman e-commerce.
- Notifikasi global: menampilkan alert atau toast dari mana pun di aplikasi.
- Data sinkron antar fitur besar: misalnya saat filter di sidebar harus mengubah tabel di halaman utama dan grafik di bawahnya.
