# Typical Use Case

## Storing Cache Strategy

Sebuah data kita misalkan dengan data role. Data tersebut bisa dikelompokkan beberapa jenis berdasarkan kegunaannya:

- `Tabulation` state menyimpan total, dan list of data dengan key `page, size, filters`
- `Select Option` state menyimpan semua user dengan berupa list of `value-label`

### Invalidation

Kedua data tersebut disimpan dengan value index pertama yang sama agar ketika ada aksi (C/U/D) akan dilakukan invalidasi.
