# Penjelasan

Pastikan tidak mengubah state dengan menggunakan `state hasil modifikasi`. Ini dikarenakan React mendeteksi perubahan lewat perbandingan referensi. Ketika kita modifikasi objek, referensinya sama, sehingga react menganggap tidak berubah, padahal nilainya berubah hanya referensi berbeda.

## Contoh Salah

```jsx
state=2
setState(state);
setState(2); //benar

state.age=4
setState(state);
setState({...state, age:4});//benar

state.push(4);
setState(state);
setState([...state, 4]); //benar
```

## Prev Param Setter

Prev param juga merupakan referensi ke obj state saat ini.

```jsx
setTodos(prev => {
  prev.push({ id: 3, text: "Belajar React" }); // âŒ mutasi langsung
  return prev; // referensi tetap sama
});
```
