# Metadata

Meningkatkan SEO dan link informatif. Ada 2 cara yaitu dengan export variable atau export fungsi

- `export metadata = {}`
- export fungsi (untuk dynamic page)

```jsx
export func async generateMetadata({params}){
    const meals = await fetch();
    if(!meals) return notFound();

    return {
        description: meals.description
    }
}
```

## Prioritas

1. export variable di page.jsx
2. export func di page.jsx
3. export variable di layout
