# Reserved Things

Dalam NextJS terdapat beberpa penamaan yang bersifat `reserved` yang akan memberikan perilaku khusus ketika sesuatu diberi nama tersebut.

## Files

<https://nextjs.org/docs/app/api-reference/file-conventions>

- `app/icon.png` favicon
- `page.js` file halaman berupa server component
- `layout.js` layout yang digunakan untuk halaman. Bisa nested, tapi pastikan hanya ada 1 `<HTML>` di root layout saja
- `error.js` Ditampilkan ketika ada segment/component error saat rendering, bisa diterapkan ke root, atau page lebih dalam.
- `not-found.js` Ditampilkan ketika url tidak terdaftar pada route
- `loading.js` Ditampilkan saat rendering server component berlangsung

## Variable

- `export const metadata` berisi object yang akan di render pada `<head>` element HTML, bisa diletakkan dalam file page atau layout.
